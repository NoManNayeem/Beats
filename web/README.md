## Deployment

This page is automatically deployed to GitHub Pages when changes are pushed to the `main` branch in the `web/` directory.

### Manual Deployment

1. Go to repository Settings
2. Navigate to Pages section
3. Select source: GitHub Actions
4. The page will be available at: `https://nomannayeem.github.io/Beats/`

## Customization

- Edit `index.html` to change content
- Modify `assets/css/style.css` for styling
- Update `assets/js/script.js` for interactive features
- Add images to `assets/images/` directory
